## Funksiyalar ro'yxati

### Asosiy funksiyalar
- `hello()` — "Hello, World!" qaytaradi
- `kvadrat(a)` — sonning kvadratini qaytaradi
- `kub(a)` — sonning kubini qaytaradi

### Arifmetik funksiyalar
- `yigindi(a, b)` — ikki son yig'indisi
- `ayirma(a, b)` — ikki son ayirmasi
- `kopaytma(a, b)` — ikki son ko'paytmasi
- `bolish(a, b)` — ikki son bo'linmasi

### Taqqoslash funksiyalari
- `max_son(a, b)` — katta sonni qaytaradi
- `min_son(a, b)` — kichik sonni qaytaradi
- `juftmi(a)` — son juftligini tekshiradi
- `toqmi(a)` — son toqligini tekshiradi
